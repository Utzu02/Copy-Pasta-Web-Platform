## **📂 Structura Proiectului**

### **Frontend**
- **Framework**: React@Vite

### **Backend**
- **Framework**: Node.js + Express  
- **Database**: MongoDB  
## **📦 Instalare**

### **Deschidere Server Frontend**  
```bash
cd front-end/copy-pasta
npm run dev
```
### **Deschidere Server Backend**
```bash
cd back-end
npm run utzu (utzu -> comandă pentru Nodemon)
```
